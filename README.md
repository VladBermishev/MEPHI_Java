# Build
```shell
./gradlew jar
```
# Run
```shell
java -jar build/libs/RK1-1.0-SNAPSHOT.jar
<or>
./gradlew run
```
# Test
```shell
./gradlew test
```

# Help
```shell
#use help command while running service, to see
Commands:
  q, quit - quit from service
  l, list - list available user links
  s, shorten - shorten link
  g, go - go to link
  h, help - list available commands
```
